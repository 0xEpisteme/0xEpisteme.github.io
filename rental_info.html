Rental Info

rentals.py

This is a small project that obtains rental information from Zillow and places the information in a pandas DataFrame object and imports the DataFrame into an Excel document.

The Excel sheet can then be analyzed or the program can be modified to obtain different results.

The program is run in the command line and takes a zip code as an argument.

The program creates an Excel document with the zip code in the title.

If information for a particular field cannot be obtained, a numpy.nan value is used instead.
